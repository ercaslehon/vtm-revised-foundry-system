# Changelog

## 0.6.6-dev

- Добавлен автозапуск мастера создания персонажа при создании нового Actor типа `vampire` через стандартный диалог Foundry.
- После создания нового персонажа система закрывает обычный лист и открывает пошаговый мастер.
- Импорт Trechkalov JSON теперь создаёт Actor с флагом подавления мастера, чтобы импортированные готовые чарники не проходили создание заново.

# 0.6.5-dev

- Убран большой блок «Создание персонажа» с чеклистами из основного листа актёра.
- Пошаговый мастер создания персонажа и расчёт чеклистов сохранены в коде, но больше не занимают место в уже готовых чарниках.
- Чарник снова сразу переходит от основных блоков к оружию, магии крови, ритуалам и заметкам без лишней бухгалтерской простыни.

# 0.6.4-dev

- Добавлен пошаговый мастер создания персонажа в отдельном окне.
- Добавлены шаги: клан и база, характеристики 7/5/3, способности 13/9/5, дисциплины/факты биографии/добродетели, свободные очки, достоинства/недостатки, итоговая проверка.
- Добавлена кнопка открытия карточки клана прямо из мастера.
- Дисциплины, факты биографии, достоинства и недостатки добавляются из каталогов внутри мастера.
- Чеклист остаётся сопровождающим и предупреждает о нарушениях, но не блокирует создание персонажа.

# Changelog

## 0.6.3-dev

- Добавлен блок «Создание персонажа» с двумя чеклистами.
- Первый чеклист проверяет базовое распределение: характеристики 7/5/3, способности 13/9/5, лимит 3 точки на способность, дисциплины 3, факты биографии 5, добродетели 7, формулы Человечности, Воли и стартовой крови.
- Второй чеклист считает свободные очки: редактируемый пул по умолчанию 15, достоинства, недостатки, стоимость характеристик, способностей, дисциплин, фактов биографии, добродетелей, Человечности/Пути и Воли.
- Добавлены кнопки «Фикс Человечности», «Фикс Воли» и «Бросить кровь d10».
- Чеклисты предупреждают о нарушениях, но не блокируют редактирование.

## 0.6.2-dev

- Свален единый общий каталог достоинств/недостатков без раздельных фильтров по редакциям.
- Расширение каталога до 376 карточек: 183 достоинства, 193 недостатка.
- Добавлены компиляционные, V20/4ed, Dark Ages и дополнительные социальные/клановые варианты.
- Все карточки снабжены описанием, системной выжимкой, эффектом, audit/source и automation notes.

# Changelog

## 0.6.1-dev

- Добавлен встроенный каталог достоинств и недостатков.
- Добавлена команда `game.vtmRevised.importBuiltInMeritsFlawsCatalog()`.
- Добавлена кнопка импорта достоинств/недостатков в директории предметов.
- Достоинства и недостатки теперь добавляются в чарник из выпадающего списка справочника.
- Название достоинства/недостатка открывает отдельную карточку с описанием, механикой, эффектом и источником.
- Для достоинств/недостатков добавлены структурированные поля эффекта: тип, цель, модификатор сложности, модификатор кубов, заметки.
- Кнопка использования открывает карточку или выводит эффект в чат.

## 0.5.2-dev

- Добавлен справочник кланов и карточка описания клана.

## 0.6.7-dev

- Добавлены карточки описания для Натуры и Маски из шапки чарника и мастера создания.
- Мастер создания теперь живо пересчитывает факты чеклиста при изменении значений.
- Максимальные значения характеристик и способностей в мастере завязаны на поколение.
- Подсчёт характеристик в мастере учитывает базовую единицу: потраченные точки = рейтинг - 1.
- При выборе клана мастер автоматически добавляет клановые дисциплины персонажу с рейтингом 0.
- В списке дисциплин мастера добавлены кнопки-книжечки для просмотра справки.
- В выборе достоинств/недостатков добавлены кнопки предпросмотра карточки выбранной черты.
- Подправлен вид кнопок добродетелей и элементов мастера создания.
